egashira.jpについて

このegashira.jpはえがしらけいすけ が管理・運営している、主にWeb関連の技術的なスニペットなどを書くブログサイトです。

ミニマルをモットーに高速なサイトを目指していて、載せているコードもできるだけミニマルな実装になるように考えたものです。
機能を実現するのにライブラリ（例えばJavaScriptではjQueryなど）を使用せずに、コードをコピーするだけで使えるようなスニペットを掲載していますので、プラグイン的に使用するのではなく内容を把握してどんな動きをしているのか理解し、自分のプロジェクトなどに使用するのに役立つと思います。

この高速なサイトを実現するのにGoogleのPaaS、Google App Engineを使用していますが、以前はレンタルサーバ上でWordpress使ったり、PHPやGolangでブログエンジンみたいなものを作って試行錯誤してきたため、記事のURLが変わったり、以前にあった記事が無くなっていることが多々あります。検索エンジンなどから飛んでこられた方が４０４ページを表示された場合はご容赦ください。
現在はサイト内の検索もできない状態ですが、そのうち検索できるようにしたいと思います。

また、このサイト自体が実験の場であったりするので、古いブラウザ対応はバッサリ捨てているほか、不具合もあると思いますが大目に見てください。
